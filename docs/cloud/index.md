# Kube-DC Cloud

Welcome to Kube-DC Cloud — a fully managed cloud platform where you can deploy virtual machines, Kubernetes clusters, and containerized workloads without managing the underlying infrastructure.

## What You Can Do

- **Deploy Virtual Machines** — Launch Linux and Windows VMs with a few clicks or via kubectl
- **Run Kubernetes Clusters** — Provision managed Kubernetes clusters (nested clusters) for your applications
- **Flexible Networking** — Get public and floating IPs, configure private networks and NAT
- **Persistent Storage** — Block storage (PVC), S3-compatible object storage, and backups
- **Team Collaboration** — Manage users, roles, and billing across your organization
