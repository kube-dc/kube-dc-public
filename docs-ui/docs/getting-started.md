# Getting Started

Welcome to the documentation! This site is built using PatternFly v5 and supports Markdown files.

## What is this?

This is a documentation website that:

- Uses **PatternFly v5** for UI components
- Renders **Markdown files** for content
- Provides easy navigation through documentation
- Supports GitHub Flavored Markdown (GFM)

## Features

### Markdown Support

You can use all standard Markdown features:

- Lists (ordered and unordered)
- **Bold** and *italic* text
- `Inline code`
- Code blocks
- Tables
- Blockquotes
- And more!
